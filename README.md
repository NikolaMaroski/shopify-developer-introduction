# Shopify Theme Developer - Interview Crash Course

An interactive, single-page learning app covering everything you need to pass a senior-level Shopify theme developer technical interview.

## Topics Covered

- **Rendering Pipeline** - How Shopify renders a page from request to HTML
- **Liquid Templating** - Objects, tags, filters with complete references
- **Theme Architecture** - File structure, layouts, templates, sections, snippets
- **Sections & Blocks** - Schema, settings, blocks, presets, section groups
- **JSON Templates (OS 2.0)** - Modern vs legacy approach
- **Theme Settings** - settings_schema.json, settings_data.json, all input types
- **Metafields & Metaobjects** - Access patterns and dynamic sources
- **Ajax API** - Cart operations and Section Rendering API
- **Performance Optimization** - Images, JS, CSS, Liquid, Lighthouse targets
- **Accessibility** - WCAG 2.0 AA requirements
- **Shopify CLI** - Commands and development workflow
- **Theme App Extensions** - App blocks vs app embeds
- **Checkout Customization** - UI extensions and Shopify Functions
- **JavaScript Patterns** - Web Components, variant handling, predictive search

## Features

- Interactive tabbed references for all Liquid objects
- Animated rendering pipeline diagrams
- Side-by-side comparisons (legacy vs modern)
- 10-question practice quiz with instant grading
- 12 interview Q&As with model answers
- Quick reference cheat sheet
- Syntax-highlighted code examples
